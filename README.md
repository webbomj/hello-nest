
## Description
Simple API for adding, deleting, updating, reading movies on the framework NestJS.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Used
1. Typescript
2. NestJs
3. ESlint
4. NodeJs
5. Jest
